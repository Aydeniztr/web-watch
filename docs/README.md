## 🎯 What You Get

### ⏰ Full-Featured Clock
- Real-time digital display
- 12/24 hour format toggle
- Current date and time
- Apple-inspired design

### ⏱️ Advanced Timer System
- Add unlimited simultaneous timers
- Custom durations (hours, minutes, seconds)
- Optional labels
- Play/Pause/Reset/Delete controls
- Visual progress bar
- Audio notifications
- Persistent storage

### 🌍 Global Market Hours Tracker
- 10 major stock exchanges monitored
- Real-time open/closed status
- Green for open, red for closed
- Timezone-aware calculations
- Updates every 60 seconds

## ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+1` | Jump to Clock tab |
| `Ctrl+2` | Jump to Markets tab |
| `Ctrl+Shift+C` | Toggle 12/24 hour format |
| `Enter` | Create timer (in modal) |
| `Esc` | Close modal |
